# Beautiful-BirthdayCounter

## Code :
A simple Birthday Counter that Subtract the current date and my birthday date :
<br>
<code>
DateTime endTime = new DateTime(2017, 08, 09, 0, 0, 0);<br>
var ts = endTime.Subtract(DateTime.Now);
</code>
